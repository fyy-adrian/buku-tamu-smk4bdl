
# Website buku tamu
Selamat datang di repositori website buku tamu, website yang di buat oleh Rafi Adrian Dan M. Riandra, yang di peruntukan untuk mencatat kehadiran tamu di SMKN 4 Bandar Lampung


## Fitur

- Menambah list tamu
- Melihat semua list tamu
- Meng-Export data ke excel


## Teknologi

**Client:** Laravel 11

**Server:** PHP, Sqlite


## Installation

1. Clone repositori ini

```bash
    git clone https://github.com/Kepoo311/buku-tamu-smk4bdl.git
    cd buku-tamu-smk4bdl
```

2. Instal dependensi

```bash
   composer install
   npm install
```

3. Buat file konfigurasi .env

4. Jalankan

```bash
    npm run dev
    php artisan migrate //untuk membuat db
    php artisan serve
```
